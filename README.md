This GUI provides users with accurate predictions of the Shear Capacity of UHPFRC Beams without Stirrups. Users can simply input the required parameters as prompted on the interface or obtain parameters from the database by clicking the "Random Parameters" button to receive the shear bearing capacity prediction.
